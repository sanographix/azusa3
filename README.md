Azusa 3
=============

Free Google Slides & Keynote Theme

- <https://azusa3.sanographix.net>

# Download & Install

## For Keynote

### System requirements

- Keynote v10.0 (for macOS)

### Install

- <https://github.com/sanographix/azusa3/archive/v3.0.zip>

1. Unzip `azusa3-3.0.zip`
2. Open `azusa3.kth`
3. Select "Add to theme selector".
